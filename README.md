# Weather App

This is a simple weather app built with Node.js, Express, Axios, and EJS. It fetches weather data from the OpenWeatherMap API based on the user's input city and displays relevant information.

## Features

- **Weather Information:** Get real-time weather data including temperature, humidity, and wind speed.
- **Dynamic Weather Icons:** Display weather icons based on the current weather conditions.
- **Error Handling:** Show user-friendly error messages for invalid cities or when there's an issue fetching weather data.

## Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/) (Node Package Manager)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/weather-app.git

## Steps to follow:

1. Change into the project directory:
   ```bash
   cd weather-app

3. Install all dependencies:
   ```bash
   npm install
4. Create a '.env' file in the root of your project and add your OpenWeatherMap API key:
   ```bash
   OPENWEATHER_API_KEY=your_api_key_here

## Usage

1. Start the server:
   ```bash
   nodemon index.js

2. Open your web browser and navigate to [http://localhost:3000](URL).
3. Enter a city name in the search box and press Enter to get the current weather information.
